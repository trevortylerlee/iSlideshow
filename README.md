# iSlideshow for macOS

Drag in images and videos and view a slideshow. Toggle shuffle and loop.

[Download on the App Store ($4.99)](https://apps.apple.com/app/islideshow/id6764747094?mt=12)

[Download latest release](https://github.com/trevortylerlee/iSlideshow/releases/latest)

## Keyboard Shortcuts

- **Open Files**: <kbd>⌘ + O</kbd>
- **Start Slideshow**: <kbd>Return</kbd>
- **Next File**: <kbd>→</kbd>
- **Previous File**: <kbd>←</kbd>
- **Pause Slideshow**: <kbd>Space</kbd>
- **Close Slideshow**: <kbd>Esc</kbd>
- **Remove All Files**: <kbd>⌘ + A</kbd> and then <kbd>⌫</kbd>
